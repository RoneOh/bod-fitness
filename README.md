# BOD 健身记录

BOD 是一个手机优先的健身记录 MVP，用于记录饮食、训练、体重和喝水量，并根据目标给出每日摄入建议、训练消耗建议、进度反馈和成就激励。

## 当前功能

- 首次进入目标引导：减脂、增肌、保持
- 今日饮食记录：选择常见食物、份量自动估算热量和蛋白质
- 支持手动修正热量，适合知道精确数值的用户
- 根据身高、体重、目标和体重趋势估算今日摄入热量
- 根据体重变化自动小幅调整后续摄入建议
- 今日训练、体重、喝水记录
- 基于 MET 的训练热量估算
- 今日摄入目标、训练目标、完成百分比、还差多少 kcal
- 今日建议和成就反馈
- 7/30 天体重趋势、本周训练热量
- 本地浏览器保存数据，不需要登录

## 本地预览

直接打开 `index.html`，或使用任意静态服务器预览。


## 隐私说明

当前版本不会上传用户数据。训练、体重、喝水和目标数据都保存在用户当前浏览器的 `localStorage` 中。用户更换设备、切换浏览器或清理浏览器数据后，记录可能丢失。
