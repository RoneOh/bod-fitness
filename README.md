# BOD 健身记录

BOD 是一个手机优先的健身记录网页应用，用于帮助健身用户记录饮食、训练、体重和喝水量，并根据目标给出每日建议。

## 功能

- 目标设置：支持减脂、增肌、保持
- 饮食记录：选择常见食物和份量，自动估算热量与蛋白质
- 自定义饮食：输入食物名称后，可按名称大致估算热量与蛋白质
- 智能训练计划：根据目标、训练水平、每周次数和器械条件生成动作与组次安排
- 完整训练日志：一次训练可记录多个动作和多组重量、次数、RPE，并自动带入历史表现
- 渐进超负荷：结合训练容量、估算力量和 RPE 给出下一次加重、加次数或减量建议
- 体重记录：记录体重并查看 7/30 天趋势
- 喝水记录：通过快捷按钮累计每日饮水量
- 今日建议：根据饮食、训练、体重趋势和目标生成行动建议
- 成就反馈：根据每日记录进度给予正反馈
- 每周总结：汇总训练次数、时长、力量容量、蛋白质完成情况和连续记录天数
- 记录管理：支持补记日期，并可编辑或删除饮食、训练与体重记录

## 数据说明

当前版本使用浏览器本地存储保存数据，不需要账号登录。数据不会上传到服务器；更换设备、切换浏览器或清理浏览器数据后，记录可能丢失。
